\# Système de Gestion des Achats et des Fournisseurs



\## Description

Application web de gestion des achats et fournisseurs développée avec Spring Boot et Angular.



\## Technologies

\- \*\*Backend\*\* : Java 17, Spring Boot 3.5.0, PostgreSQL, Swagger

\- \*\*Frontend\*\* : Angular, TypeScript

\- \*\*Déploiement\*\* : Docker



\## Lancer avec Docker

```bash

git clone https://github.com/imenmanou-5/achat-fournisseur.git

cd achat-fournisseur

docker-compose up --build

```



\## Accès

| Service | URL |

|---|---|

| Frontend | http://localhost |

| Backend | http://localhost:8081 |

| Swagger | http://localhost:8081/swagger-ui/index.html |
